# Road Safety Efficiency Across Indian States: A DEA-Based Analysis

This repository contains the data, code, and results for a DEA-based analysis of road safety efficiency across Indian states.

## Overview
This study applies an input-oriented Data Envelopment Analysis (DEA) model to evaluate how efficiently Indian states manage road safety given exposure levels.

## Methodology
- DEA Model: Input-Oriented, Constant Returns to Scale (CRS)
- Inputs:
  - Accidents per lakh population
  - Accidents per 10,000 vehicles
- Output:
  - Safety score (1 / total accidents)

## Key Findings
- Only Nagaland lies on the efficiency frontier
- Strong scale effect observed
- Large states show very low efficiency
